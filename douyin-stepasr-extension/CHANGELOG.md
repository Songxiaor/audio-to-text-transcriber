# CHANGELOG

## 3.0.2 (2026-06-18)

### 发布整理

- 对齐 `manifest.json`、README、用户安装说明和变更记录中的版本号。
- 补充飞书多维表格同步的数据流和隐私披露。
- 清理发布包内容，排除浏览器生成的 `_metadata`、内部进度文件和发布操作说明。
- 加强发布校验：检查文档版本一致性、隐私披露、发布包排除项和敏感文件排除项。

## 0.2.0 (2026-06-18)

### 改进

- 转写按钮改为单步操作：点击后自动检测媒体，成功后继续转写。
- 转写中在浮窗结果区显示 SSE 流式文本；无流式增量时显示动态进度提示。
- 转写中主按钮切换为「取消」，可中断当前请求。
- 首次使用且未配置 API Key 时，侧边栏自动展开「首次使用」和「API 设置」。
- 历史记录布局改为 Grid 剩余空间自适应，后处理面板展开/折叠时列表高度自动调整。
- 浮窗侧栏入口改为「☰ 记录」。
- 侧边栏版本 badge 从 `manifest.json` 动态读取版本号。
- 历史记录工具栏常驻导出入口，支持 TXT / Markdown / JSON。
- 浮窗结果和历史记录卡片显示字数统计。
- 侧边栏支持 Ctrl/Cmd+C 复制聚焦历史卡片文案，Escape 退出多选模式。
- 后处理选项保存到 `chrome.storage.local`，刷新后恢复。

### 修复

- 删除历史列表高度中的 `calc(100vh - 424px)` 魔法数字。
- 统一新增控件的暗色模式覆盖和按钮禁用态表现。
- 将 content 浮窗颜色收敛到局部 CSS 变量，降低后续样式不一致风险。
